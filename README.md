## Задачи
- Обучить агентов играть в `NeuralMMO`, которая была создана как площадка для проведения соревнований между командами, которые управляются алгоритмами `RL`.
- Принять участие в соревновании `NeurIPS 2023: NeurlMMO Challenge`.
## Описание игры
- команда из `8 агентов` собирает еду и воду, чтобы выжить.
- каждый агент имеет `8 навыков`, можно развивать.
- `ресурсы` восстанавливают еду, воду и здоровье, создают боеприпасы для боя.
- `агенты` могут атаковать друг друга, приобретать доспехи и оружие, торговать предметами на рынке.
- также в мире есть неигровые персонажи, которых можно победить для получения предметов и увеличения силы.

## Содержание
- `docker` - папка с файлами для создания нужных зависимостей.
- `tools` - чекпоинты, логи и графики обучения модели.
- `torchbeast` - основные классы и функции.
- `main.py` - использование кода из `torchbeast` и дополнительных wrapper-ов для запуска обучения.
- `model.py` - энкодер
- `utils.py` -  дополнительные парсеры и wrapper-ры.
